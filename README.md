## Smart Password Strength Checker

A modern and interactive password strength checker that helps users create safer passwords while learning good security practices.

This project not only evaluates password strength but also provides suggestions, visual feedback, and helpful tips to improve password security.

---

##  Key Features

-  **Real-time Password Strength Detection**  
  Instantly analyzes password strength as the user types.

-  **Visual Strength Indicator**  
  A dynamic strength bar that changes color depending on password security.

-  **Password Length Indicator**  
  Displays the current password length to encourage stronger passwords.

-  **Suggested Strong Password Generator**  
  Automatically generates a strong password if the entered one is weak.

-  **Copy Suggested Password**  
  Users can copy the generated password directly to the clipboard.

-  **Dark Mode Toggle**  
  Allows switching between light and dark UI for better user experience.

-  **Password Security Tips Page**  
  Includes helpful guidelines on creating secure passwords.

-  **Show / Hide Password Option**  
  Lets users toggle password visibility while typing.
